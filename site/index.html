你的 GitHub 账号，你的授权

// 三步完成接入

1

添加连接器

ChatGPT → 设置 → Connectors → 新建 App，Server URL 填 https://github-connector.jason1105.uk/mcp。

2

用 GitHub 登录并授权

Authentication 选择 OAuth；点击 Create 后会跳转到 GitHub 登录页，用你自己的账号登录并授权。

3

直接在对话里下指令

例如"列出这个仓库的文件"或"把这个 issue 转成 PR"，ChatGPT 会调用 github-connector 完成操作。
